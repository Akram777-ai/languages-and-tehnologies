ОТЧЁТ ПО ЛАБОРАТОРНОЙ РАБОТЕ

Тема работы

Логирование, обработка ошибок и модульное тестирование в Python. Создание основного приложения, лог-файла и набора тестов.

Цель работы
	1.	Научиться применять встроенный модуль logging для записи действий программы в файл.
	2.	Реализовать функции с проверками входных данных и корректной обработкой ошибок.
	3.	Создать модульные тесты с использованием unittest.
	4.	Научиться структурировать проект: основной код, тесты, лог-файл.

⸻

Ход работы

1. Создание файла main.py

Были реализованы функции:
	•	calculate_average() — вычисление среднего значения с проверкой типов.
	•	determine_grade_letter() — перевод числовой оценки в буквенную.
	•	student_report() — формирование итогового отчёта о студенте.
	•	parse_grades_from_string() — преобразование строки в список оценок.

Добавлено логирование:
	•	запись в app.log всех шагов программы: ошибки, успешные операции, обработка исключений;
	•	формат: дата, уровень, сообщение.

В конце — интерактивный запуск через if __name__ == "__main__":.

⸻

2. Настройка логирования

Использована конфигурация:
	•	файл: app.log;
	•	уровень: INFO и выше;
	•	фиксируются ошибки, информация о расчётах, действия пользователя;
	•	заполняются при каждом запуске программы или тестов.

⸻

3. Создание тестов (test_main.py)

Использован модуль unittest.

Покрыты тестами:
	•	корректность среднего значения;
	•	работа округления;
	•	ошибки при пустом списке;
	•	ошибки типов;
	•	правильное определение буквенной оценки;
	•	корректная генерация отчёта;
	•	проверка лог-файла;
	•	корректное парсирование строки.

Каждая функция протестирована минимум 3–5 тестами.
